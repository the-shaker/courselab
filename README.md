# Курсовая работа по дисциплине «Фронтенд-разработка» 

Клиентская часть веб-панели для управления контентом онлайн-курсов: каталог
курсов, статистика платформы, тикеты поддержки и пользователи.

## Стек

**Клиент (`app/`)**
- Vue 3 (Composition API, `<script setup>`)
- Vite — сборщик и dev-сервер с HMR
- Vue Router 4 — HTML5 History API, без хешей
- CSS, `<style scoped>` + методология БЭМ
- Bun — менеджер пакетов и раннер скриптов

**API (`api/`)**
- Go + Fiber, отдаёт справочные данные в JSON
- CORS-middleware с белым списком origin-ов

**Инфраструктура**
- Docker Compose из двух сервисов (`api`, `nginx`)
- Nginx как reverse-proxy и веб-сервер статики
- HTTPS-сертификаты от Let’s Encrypt (certbot)
